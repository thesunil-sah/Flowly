Database & auth system

Sign up, log in, reach a guarded dashboard. Postgres schema (accounts / sites / subscriptions) with Alembic, argon2 password hashing, short-lived JWT access + refresh, a require_user guard, login rate-limiting, plus Google/GitHub OAuth and email verification.

JWT + refresh argon2 OAuth Alembic
Shipped
02

Tracking script

A sub-2 KB vanilla-JS snippet that reads its data-site, fires a pageview via sendBeacon to /collect, follows SPA navigations, and can never throw or break the host page.

zero-dep vanilla JS sendBeacon SPA-aware < 2 KB
Shipped
03

Ingestion pipeline

Events validated, anonymised, and stored durably. POST /collect returns 202 in milliseconds; a cookieless visitor hash (IP + UA + daily salt), bot filtering, per-site rate limiting, a Redis Stream buffer, and a batch writer draining into ClickHouse with geo/device enrichment.

cookieless hash Redis Streams ClickHouse bot filter
Shipped
04

Real-time live traffic

The headline feature: visitors on screen right now. A Redis presence ZSET + pub/sub feed behind an authed, ownership-checked WS /live/{site_id}, driving a live counter, event feed, and current-pages view.

WebSocket Redis pub/sub presence ZSET
Shipped
05

Dashboard metrics

The core historical reports — overview, time-series, sources + UTM, audience (geo / device / browser / OS), and top/entry/exit pages. Sessions, bounce, and duration derived in SQL; every query ownership-scoped; a Recharts UI with date presets and period comparison.

ClickHouse SQL Recharts ownership-scoped
Shipped

Site onboarding
